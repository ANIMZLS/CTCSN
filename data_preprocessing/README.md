# DCSN: Training Dataset Generation
DCSN: Deep Compressed Sensing Network for Efficient Hyperspectral Data Transmission of Miniaturized Satellite

# Dependencies

Hyperspectral images of AVIRIS from 'https://aviris.jpl.nasa.gov/dataportal/'.

# Generating dataset

- To generate dataset: run 'generate_data.m' using matlab.

- Use 'generate_data.m' by matlab and refer to 'LoadHypercubesMatlab.pdf' to generate dataset.

- Generate the txt files with the file path of the training/testing dataset by '../file_path.py', such as'../train_4fig.txt, ../val_4fig.txt'.
